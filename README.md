# 키보드 쇼핑몰 프로젝트  
> 활용 기술 : vue3





**프로젝트 상세**  
이 프로젝트는 개인 프로젝트로 다크모드, 라이트모드 및 데이터 출력을 구현했습니다.

  - **다크모드 라이트모드**: data-theme 속성을 이용한 다크모드 라이트모드 토글 기능 구현 
  - **데이터 동적 출력**: 임시 데이터를 이용해 데이터를 동적으로 출력했습니다.
